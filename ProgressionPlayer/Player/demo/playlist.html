Setting up Playlist

Step 1: Create the playlist:

Create the playlist with HTML. The key class that is unique from the other players is "progression-playlist". Also, the title="" tag displays the title in the playlist.

You can adjust the playlist height via the inline style of style="padding-bottom:174px;".

<div class="progression-playlist-height responsive-wrapper responsive-audio" style="padding-bottom:174px;"><!-- Adjust padding-bottom as needed so the playlist doesn't cover content below -->
<audio class="progression-playlist progression-skin progression-audio-player" controls="controls" preload="none">
	<source src="../media/AirReview-Landmarks-02-ChasingCorporate.mp3" title="Cheer Up by sevenfoursounds" type="audio/mpeg"/>
	<source src="../media/indie-rock.mp3" title="Indie rock by AdamCaptured" type="audio/mpeg"/>
	<source src="../media/winner-preview.mp3" title="Winner by kristijanf" type="audio/mpeg"/>
	<source src="../media/get-out-of-my-way-preview.mp3" title="Get Out of My Way by AdamCaptured" type="audio/mpeg"/>
	<source src="../media/positive-thinking-preview.mp3" title="Positive Thinking by BeatheBeat" type="audio/mpeg"/>
	<source src="../media/glitchy-digital-preview.mp3" title="Glitchy Digital Crystal Logo #07 by alkis" type="audio/mpeg"/>
</audio>
</div><!-- close .responsive-wrapper -->

Step 2: Start Playlist via JavaScript

<script>
$('.progression-playlist').mediaelementplayer({
	audioWidth: 400, // width of audio player
	audioHeight:40, // height of audio player
	startVolume: 0.5, // initial volume when the player starts
	loop: false, // useful for <audio> player loops
	features: ['playlistfeature', 'prevtrack', 'playpause', 'nexttrack','current', 'progress', 'duration', 'volume', 'playlist'],
	playlist: true, //Playlist Show
	playlistposition: 'bottom' //Playlist Location
	});
</script>

Audio Playlist



Optional: Playlist Closed at start

If you want a playlist closed at the start, you will need to set "playlist: true, //Playlist Show" to false.

In addition you will need to change the inline padding-bottom to 0px;.

Lastly, you will edit /css/progression-player.css under section "02. Easy Customization". You will add in the height of your playlist. Example: 

.active-progression {padding-bottom:174px !important;  }
